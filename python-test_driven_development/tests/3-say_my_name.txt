How to Use 3-say_my_name.py
==============================

This module defines a function that like the Destiny's Child will say my name. 

Usage
=====

``say_my_name(...)`` returns a name

>>> say_my_name = __import__('3-say_my_name').say_my_name
The two variables accepting string
    >>> say_my_name("Jack", "Ryan")
    my name is Jack Ryan

First name attribute is a string but last name is not
    >>> say_my_name("Jack", 42)
    Traceback (most recent call last):
    ...
    TypeError: last_name must be a string

First name attribute not a string but last name is.
    >>> say_my_name(27, "Ryan")
    Traceback (most recent call last):
    ...
    TypeError: first_name must be a string

If both the first and last name arguments are not entered
    >>> say_my_name()
    Traceback (most recent call last):
    ...
    TypeError: say_my_name() missing 1 required positional argument: 'first_name'
